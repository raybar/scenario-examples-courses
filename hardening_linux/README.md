# Hardening Linux: Fortaleciendo tu Servidor Ubuntu

## Descripción

En este escenario interactivo, aprenderás las técnicas fundamentales para fortalecer la seguridad de un servidor Ubuntu. A través de una serie de comandos prácticos, implementarás medidas de seguridad clave para proteger tu sistema contra amenazas comunes y reducir la superficie de ataque.

## Objetivos de aprendizaje

* Asegurar el acceso SSH.
* Configurar un firewall básico con UFW.
* Actualizar y asegurar paquetes del sistema.
* Configurar auditorías básicas del sistema.
* Implementar prácticas de seguridad de contraseñas.
* Entender la importancia de la gestión de usuarios y grupos.

## Duración estimada

45-60 minutos
